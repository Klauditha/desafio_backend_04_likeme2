### Ejecución

- Para ejecutar proyecto BackEnd
    ```
    cd .\BackEnd\
    npm run dev
    ```

- Para ejecutar proyecto FrontEnd
    ```
    cd .\FrontEnd\
    npm run dev
    ```

Seguir la estructura de conexión propuesta en el archivo env.example dentro del proyecto BackEnd.
